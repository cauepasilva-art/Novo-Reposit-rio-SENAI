<h2>📝DESCRIÇÃO DO PROJETO📝</h2>
<p>Servidor Backend, onde é possivel realizar dos tipos de requisições HTTP</p>
<h2>Método GET</h2>
<p>O objetivo do método GET é simplesmente recuperar dados do servidor. O método GET é usado
pra solicitar qualquer um dos seguintes Recursos:

<li>-Uma página da web ou arquivo HTML</li>
<li>-Uma imagem ou vídeo</li>
<li>-Um documentos JSON</li>
<li>-Um arquivo CSS ou arquivo JavaScript</li>
-Um arquivo XML</p>
<h2>Método POST</h2>
<p>O método de solicitação POST HTTP envia dados ao servidor para processamento.
Os dados enviados ao servidor normalmente estão no seguinte formato:
<li>-Campos de entrada de formulários online.</li>
<li>-Dados XML ou JSON</li>
<li>-Dados de texto de parâmetros de consulta</li>
